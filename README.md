🎉 一个支持 `vue2.x / vue3.x` 的抽奖插件  
🎨 `奖品 / 文字 / 图片 / 颜色 / 按钮` 均可配置  
🏅 支持`同步 / 异步`抽奖，概率`前 / 后端`可控  
🚀 自动根据`dpr`调整清晰度, 并支持`rem`适配移动端
