# 在 Vue 中使用

# 方式 1：通过 import 引入

# 1. 首先安装插件

# npm 安装
npm install @lucky-canvas/vue@latest

# 或者 yarn 安装
yarn add @lucky-canvas/vue@latest

友情提示

极少的情况下, 低版本的脚手架 (或自己搭建的webpack项目) 可能会出现安装/打包失败

目前推测可能是依赖有问题导致的, 建议删除 node_modules 和 **-lock.json 这两个文件

然后再次尝试 npm install 安装所有依赖

如果还是不能使用, 那还有一个最终方案, 就是使用下面的CDN链接 (经过验证, CDN链接可以避免依赖问题)


# 方式 2：通过 script 标签引入

CDN 链接：


  • Vue2.x：https://cdn.jsdelivr.net/combine/npm/@vue/composition-api@1.2.4,npm/vue-demi@0.11.4/lib/index.iife.min.js,npm/@lucky-canvas/vue@0.0.8

  • Vue3.x：https://cdn.jsdelivr.net/combine/npm/vue-demi@0.11.4/lib/index.iife.min.js,npm/@lucky-canvas/vue@0.0.8

URL有点长不用担心, 因为是个合并链接, 就正常使用即可 (记得要在你的vue脚本后面引入)
