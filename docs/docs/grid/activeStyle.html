中奖标记目前可支持的属性较少

  • activeStyle?: object 中奖标记样式

    • fontColor?: string 字体颜色

    • fontSize?: string 字体大小(px)

    • fontStyle?: string 字体样式

    • fontWeight?: string 字体粗细

    • background?: string 格子的背景颜色 （默认是 '#ffce98' 橘黄色）

    • shadow?: string 格子阴影 （由 4 个值组成：1.水平位置、2.垂直位置、3.模糊度、4.阴影颜色）

# 关于中奖标记样式

  • activeStyle配置的样式只有在中奖标记滑过时才会生效，如果没有开始抽奖，中奖标记默认不显示
  • 中奖标记会以索引(奖品数组的index)升序的方式进行游走，所以index的顺序就决定了九宫格该如何旋转
