自由化

奖品 / 文字 / 图片 / 颜色 / 按钮均可自由配置；支持同步 / 异步抽奖；中奖概率前端 / 后端可控；

多端适配

一个支持 Js / JQuery / Vue / uni-app / 微信小程序 的抽奖插件；

React 版本还在开发中...

响应式

自动根据设备 dpr 适配清晰度；并支持使用 百分比 / rem / rpx 属性来调整移动端布局；
