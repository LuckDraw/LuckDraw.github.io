# 关于转盘的偏移角度

为什么要有 offsetDegree 属性?

转盘的指针指向正上方, 默认是指向缝隙区域, 这也就出现了两种情况


# 1. 指针默认指向奖品

指针默认指向缝隙 指针默认指向奖品

offsetDegree = 0

offsetDegree = 360 / 6 / 2


# 2. 奖品图片和背景图片

如果你发现 UI 给到的背景图, 就是缝隙朝着正上方, 那你就不需要调整 offsetDegree

反之, 如果你发现背景图是奖品朝上的, 那你就需要调整offsetDegree属性
