# 中奖标识没有停在正确的位置

  • 如果你是最新版请忽略, 最新版无任何致命 bug, 请放心食用

抱歉, 这是一个 bug

目前已修复, 请保持核心依赖 lucky-canvas 的版本大于等于 1.7.7

如何更新? -> 官网 -> 使用 -> 在 xxx(你的框架) 中使用 -> 按照流程重新安装一次

复现问题的场景

  1. 低版本的抽奖插件中, 在减速阶段切换后台
  2. 直到抽奖结束后再切换回抽奖页面, 会发现中奖标识没有停留在正确的位置

解决方案

更新到最新版, 各个端的最新版均已修复该问题

升级的过程中需要注意以下两个属性的变动

  • 大转盘: 指针默认停在第一个奖品上 -> 默认停在奖品缝隙上
  • 大转盘: 指针默认在扇形内随机停止 -> 默认停在扇形中线上
