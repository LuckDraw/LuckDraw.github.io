默认配置 - defaultConfig

defaultConfig是对抽奖插件进行整体的配置

  • default-config?: object 默认配置

    • gutter?: number 格子之间的间距 （默认等于 5）

    • speed?: number 旋转速度峰值 （默认为 20，建议配置范围 10 ~ 30）

    • accelerationTime?: number 加速时间 （单位为毫秒，默认等于 2500 毫秒）

    • decelerationTime?: number 减速时间 （单位为毫秒，默认等于 2500 毫秒）

# 关于旋转速度

旋转总共分为三个阶段：

  • 第一阶段accelerationTime进行加速旋转，速度会从 0 加速到speed设置的值（默认为20），加速曲线恒定的情况下，你设置的时间越短，加速度越大
  • 第二阶段处于加速到峰值之后 && 调用stop方法之前，此时加速度恒定为speed
  • 第三阶段decelerationTime会缓慢停止，在你调用stop方法之后，速度会从speed逐渐降低到 0

    • 注意：当你接口的请求速度，小于等于accelerationTime时会跳过第二阶段
